## Amaze-On
Amaze-Player is a video library where user can see reviews of mobiles & laptops.

## Preview
https://user-images.githubusercontent.com/72139503/169534117-8ee1b813-9b84-46c4-9aba-a3a5778894f6.mp4

## Tech Stack

**Client** - ReactJS , JWT

**Hosted** - Netlify



## Features

- Videolisting Page
- Like Management
- Watchlater Management
- History Management
- Playlist Management
- Single Video Page
- Login / Signup
